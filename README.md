# CRUD Practice Project MERN

This is a simple CRUD practice frontend project built using React and Vite. The project focuses on understanding React basics, component structure, form handling, state management, and GitHub workflow.

---

# Features

- Product Form UI
- React Functional Components
- useState Hook
- Form Handling
- Dynamic Input Handling
- Responsive Layout

---

# Tech Stack

- React JS
- Vite
- JavaScript
- CSS
- Git & GitHub

---

# Installation

Clone the repository:

```bash
git clone https://github.com/usamazeeshan/Crud-Practice-Mern.git
````

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```

---

# Learning Objectives

This project helped in learning:

* React basics
* Components
* Props
* State management using useState
* Event handling
* Form submission
* Git and GitHub basics
